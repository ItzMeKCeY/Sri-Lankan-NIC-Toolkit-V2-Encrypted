## 🌌 Overview
An elegant, ultra-high-performance, and completely secure client-side web application designed to generate, validate, decode, and bulk-process Sri Lankan National Identity Cards. 

> 🌀 **Design Philosophy:** 𓏲๋࣭࣪˖ Built with a modern glassmorphism UI token system, providing a premium dark/light mode experience tailored for next-generation usability.

## 🚀 Key Features

* **🔮 NIC Generator** `⋆˚࿔`
  * Instantly generates all possible real 12-digit NIC permutations for any given Date of Birth and Gender.
* **🔎 Structural Decoder** `✦`
  * Extracts deep analytics (Birth Year, precise Birth Date, Gender, Age in 2026, Format Era, and Serial Index).
* **📊 Bulk Processor (.CSV)** `ᝰ.ᐟ`
  * Drop a CSV list of NICs to instantly validate mathematical consistency and auto-download a verified report.
* **🔒 Obfuscation Core** `ִֶָ☾.`
  * The production logic is tightly compressed and polymorphic-packed to neutralize basic source-scraping and unauthorized `Web2Zip` cloning.
