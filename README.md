# 🇱🇰 Advanced SL NIC Toolkit V2

An elegant, high-performance, and completely secure client-side web application designed to generate, validate, decode, and bulk-process Sri Lankan National Identity Cards (both 9-digit legacy and 12-digit modern formats). 

Built with HTML5, Vanilla JavaScript, and an ultra-modern Tailwind-inspired glassmorphism UI.

## ✨ Features
* **NIC Generator:** Instantly generate all possible real 12-digit NIC permutations for any specific Date of Birth and Gender.
* **NIC Decoder:** Extract detailed analytics from a single NIC number (Birth Year, exact Birth Date, Gender, Age in 2026, Format Era, and Serial Index).
* **Bulk Processor (.CSV):** Upload a CSV list of NICs to validate mathematical structure consistency and download an automated report instantly.
* **100% Client-Side:** No database or server-side communication. All operations are processed in the user's browser, ensuring absolute data privacy.
* **Secure Obfuscation:** The core JavaScript logic is fully obfuscated to prevent easy scraping or unauthorized cloning.

## 🛠️ Tech Stack
* HTML5 (Semantic Structure)
* CSS3 (Custom Glassmorphism Design Tokens, Responsive UX)
* Vanilla JavaScript (ES6+ Runtime Obfuscated Performance)
